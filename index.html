I made this website for 2 reasons:

  1. To have a binder that I can look back on for the projects I've worked on along the way.
  2. To teach myself how to code a website, figuring those skills might come in handy someday.

I spent a few weeks diving into HTML, CSS, and JavaScript by following The Odin Project, and, like all my projects, this site is a constant work in progress. While I’m not aiming to be a frontend software engineer, I think knowing a bit about web development is at the very least, a pretty cool skill to have.

Personal Projects

These are the projects I have worked on from design conception to final realization.

(Ongoing) 2-Axis Gantry Chess Board

Currently, I am designing a chess board where the pieces move on their own. The basic mechanism is the same as what is used on laser cutters, or 3D printers.

4DOF Robotic Rover Arm

As one of the founding members of the Schulich Space Rover team, I led a group of students to create the team’s first remotely operated 4DOF robotic arm. We went on to win 3rd place in Canada on our first competition attendance.

Handheld Game Console

Running retropi on a Raspberry Pi 4, I made my friend a portable retro game emulator with 3D printing and soldering.

FRS/GT86/BRZ Project Car

My brother and I have worked on and modified this car since 2017. From supercharging the engine to changing the exhaust headers, I learnt a lot working on the FRS as a project car.

Space Invaders / Asteroids Game Mash

On my path to learning python and object-oriented programming, I developed a unique game with fully custom assets which can be found on my github. Someday, I will learn how to implement it directly on this website.
