RetroPie Handheld Game Console

Project Goals

  • Design a game console running RetroPie
  • Human centric portability
  • $80 budget + Raspberry Pi 4

Major Roadblocks

  • Raspberry Pi was not booting
    • Used root cause analysis to determine issue was due to shorted microSD card reader
    • Redesigned the console to boot through USB

Final Outcomes

  • Built a handheld game console emulating retro games
  • Revising design to be lighter and smaller
  • Gave this version to my friend for their birthday
